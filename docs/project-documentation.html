üîë Authentication

Semua endpoint API memerlukan autentikasi melalui JWT Bearer token yang diperoleh dari OAuth2 authorization server.

# Contoh request dengan authentication curl -H "Authorization: Bearer YOUR_JWT_TOKEN" \ -H "Content-Type: application/json" \ https://your-api-url/api/v1/bookings

üìã Main Endpoints

GET /api/v1/bookings

Mendapatkan daftar booking dengan pagination dan filtering

POST /api/v1/bookings

Membuat booking baru dengan validasi business rules

GET /api/v1/bookings/{id}

Mendapatkan detail booking berdasarkan ID

PUT /api/v1/bookings/{id}/status

Update status booking (BOOKED ‚Üí CHECKED_IN ‚Üí CHECKED_OUT)

GET /api/v1/rooms

Mendapatkan daftar kamar dengan availability checking

GET /api/v1/rooms/available

Mencari kamar yang tersedia berdasarkan tanggal
